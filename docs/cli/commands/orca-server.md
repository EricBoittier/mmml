# `mmml orca-server`

Persistent JAX server for ORCA.


## Usage

```bash
mmml orca-server --help
```

!!! note
    No `build_parser()` hook — see module docstring or run the command without arguments for usage.

Implementation: `mmml.interfaces.orca_external.server`
